# 前端启动指南

## 快速启动

### 1. 安装依赖（首次运行需要）
```bash
cd frontend
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问系统
打开浏览器访问：**http://localhost:3000**

## 前端功能页面

启动后，你可以访问以下页面：

1. **线索管理** - http://localhost:3000/lead
   - 线索列表查询
   - 新增、编辑、删除线索
   - 批量导入、导出
   - 搜索和筛选

2. **监控指标管理** - http://localhost:3000/monitor
   - 配置监控时间范围
   - 选择活跃机构
   - 设置活跃次数阈值
   - 可视化配置预览

3. **推送频率管理** - http://localhost:3000/push
   - 监控指标选择
   - 频率策略可视化（热力图）
   - 精细设置规则
   - 预览与测试

4. **高潜线索营销** - http://localhost:3000/highPotential
   - 统计信息展示
   - 高潜线索列表
   - 批量外呼、批量短信
   - 线索状态管理

## 注意事项

### 确保后端已启动
前端需要后端 API 支持，请确保：
- 后端服务运行在 `http://localhost:8080`
- 数据库已初始化
- API 接口可以正常访问

### 如果遇到问题

1. **端口被占用**
   - 修改 `vite.config.js` 中的 `port: 3000` 为其他端口

2. **无法连接后端**
   - 检查后端是否正常运行
   - 检查 `vite.config.js` 中的代理配置
   - 查看浏览器控制台错误信息

3. **依赖安装失败**
   ```bash
   # 清除缓存后重新安装
   npm cache clean --force
   npm install
   ```

4. **页面空白或报错**
   - 检查浏览器控制台（F12）
   - 确认后端 API 可以正常访问
   - 检查网络请求是否成功

## 开发模式

前端使用 Vite 开发服务器，支持：
- 热模块替换（HMR）
- 快速刷新
- 自动代理后端 API

## 构建生产版本

```bash
npm run build
```

构建后的文件在 `frontend/dist` 目录中。
